# 🚀 Portfólio Docker

Projeto de portfólio pessoal containerizado com Docker.

## 🐳 Tecnologias utilizadas

- Docker
- Docker Compose
- Nginx
- HTML + CSS

## ▶️ Como executar

### Build da imagem
docker build -t portfolio-docker .

### Executar container
docker run -p 8080:80 portfolio-docker

Ou com Docker Compose:
docker-compose up -d
